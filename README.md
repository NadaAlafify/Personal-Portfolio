# My Personal Portfolio

##  Description
Welcome to my personal portfolio website! This project is designed and built from scratch to serve as my digital resume and showcase. It highlights my skills, featured web development projects, and provides direct ways to connect with me.

##  Live Demo
You can view the live website here: (https://nadaalafify.github.io/Personal-Portfolio/)

##  Tech Stack
- **HTML5:** Semantic architecture to ensure a clean structural base.
- **CSS3:** Custom responsive styling featuring a modern Dark Mode theme, custom typography, and hover effects.
- **JavaScript:** Implemented for seamless interactions and dynamic components.

##  Key Sections & Features
- **Hero Section:** A welcoming introduction featuring my profile image and professional title.
- **Navigation:** Quick links to explore my **Skills** and **Projects**.
- **Social Integration:** Direct interactive links to my professional profiles including GitHub and LinkedIn.
- **100% Manual Code:** Built completely by hand without heavy layout frameworks to focus entirely on clean, maintainable CSS and semantic markup.
